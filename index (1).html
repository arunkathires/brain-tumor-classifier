Abstract
Brain tumor diagnosis from MRI scans is a critical and time-sensitive clinical task. Manual interpretation by radiologists is prone to fatigue and inter-observer variability, motivating the need for reliable automated systems. In this work, we propose a novel deep learning framework that integrates a pre-trained Vision Transformer (ViT-B/16) as a feature extractor with a Kolmogorov-Arnold Network (KAN) classification head for fine-grained tumor classification across 46 classes spanning multiple tumor types and MRI modalities (T1, T1C+, T2). To address clinical interpretability requirements, we incorporate two complementary Explainable AI techniques — GradCAM and Attention Rollout — that highlight diagnostically relevant regions. Our model achieves 88.98% top-1 accuracy and 99.23% top-5 accuracy, outperforming the standard ViT + MLP baseline across all evaluation metrics.

Dataset

We use the Brain Tumor MRI Images dataset (fernando2rad, Kaggle), containing T1, contrast-enhanced T1 (T1C+), and T2 weighted MRI images across multiple tumor categories. The dataset is split 70/15/15 for training, validation, and testing with a fixed random seed for reproducibility.

Architecture

The proposed model uses ViT-B/16 pre-trained on ImageNet as a frozen feature extractor (first 4 blocks frozen), with a two-layer KAN head replacing the standard MLP classifier. The KAN uses learned univariate functions (SiLU base + spline residual) instead of fixed linear weights.

Explainability Methods

Two complementary XAI methods are applied post-prediction to provide spatially grounded explanations for each classification decision.

GradCAM

Computes gradient of the class score with respect to the final attention block's feature maps. Produces a coarse heatmap indicating which spatial regions most influenced the prediction. Red regions indicate high relevance.

Attention Rollout

Propagates raw attention weights through all 12 transformer blocks using matrix multiplication, accounting for residual connections. Produces a per-patch relevance map from the CLS token's perspective.
